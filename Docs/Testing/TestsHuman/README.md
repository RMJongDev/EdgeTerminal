# Tests — Human

Handmatige test-scripts. Bedoeld voor een niet-technische stakeholder die zonder hulp door de flow kan lopen.

## Naming
```
<flow-naam>.md
```

## Format
- **Flow:** korte titel
- **Doel:** wat ga je testen
- **Wie kan dit uitvoeren:** rol / persona
- **Wat heb je nodig:** account, testdata, browser
- **Stappen** (genummerd, één concrete actie per stap)
- **Verwacht resultaat per stap**
- **Bij afwijking melden aan:** contactpersoon

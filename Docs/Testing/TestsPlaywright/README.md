# Tests — Playwright

Geautomatiseerde end-to-end testen. Eén bestand per kernflow.

## Naming
```
<flow-naam>.spec.ts        # script
<flow-naam>.plan.md        # plan / scenario-beschrijving (optioneel naast script)
```

## Format `.plan.md`
- **Flow:** korte titel
- **Doel:** wat verifieert deze test
- **Gebruiker / rol:**
- **Pre-condities:**
- **Stappen:**
- **Verwacht resultaat:**

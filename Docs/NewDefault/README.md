# NewDefault

Deze map bevat NewDefault-huisstijlinformatie als losse documentatiebron.
Het is bewust geen actieve skill, zodat klantstijlen en projectstijlen niet
automatisch met NewDefault-regels conflicteren.

Gebruik deze map alleen wanneer expliciet om NewDefault-styling wordt gevraagd,
bijvoorbeeld voor een voorstel, agency-cover, presentatie, interne template of
co-branded bijlage.

## Bestanden

- [brand-notes.md](brand-notes.md) - compacte samenvatting van de stijlregels.
- [design-system.html](design-system.html) - visuele preview van kleuren,
  typografie, logo's en voorstelcomponenten.
- [assets/](assets/) - opgeschoonde logo-assets met consistente bestandsnamen.
- [sources/](sources/) - originele aangeleverde bronbestanden.

## Prioriteit

Voor klantwerk geldt:

1. Klantstijl wint voor klantgerichte output.
2. Projectspecifieke designregels winnen voor app-UI en productwerk.
3. NewDefault-stijl wordt alleen gebruikt als de gebruiker daar expliciet om
   vraagt of als NewDefault duidelijk de afzender is.
